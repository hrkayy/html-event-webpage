# HTML Event Webpage

A static event website built using pure HTML files to showcase an event layout with a home page, upcoming events, and a booking page.

## 📄 Pages Included

- **home.html** – Main landing page  
- **upcoming.html** – List of upcoming events  
- **book.html** – Booking page/form  
- **event.jpg** – Event image used in the design  
*(All pages are built with basic HTML — no additional frameworks.)* 

## 🚀 Live Preview

You can preview this website by opening any of the `.html` files in your browser. It’s a simple static site — no server required!

## 🛠️ Built With

- HTML  
*(Static event webpage without CSS/JS)*
